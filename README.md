🛍️ Luxora – Full Stack E-Commerce Website

Luxora is a modern full-stack eCommerce web application designed to simulate a real online shopping platform. The project allows users to browse products, view details, and interact with a dynamic backend powered by REST APIs.

It demonstrates practical implementation of frontend development, backend APIs, and database integration, making it a complete full-stack project.

🚀 Features

Product listing and product detail pages

Dynamic backend APIs

RESTful architecture using Express

MongoDB database integration

Responsive modern UI

Cross-origin communication with CORS

Scalable backend structure

⚙️ Installation

Clone the repository

git clone https://github.com/yourusername/luxora.git

Go to project directory

cd luxora

Install backend dependencies

npm install

Start the server

node server.js
